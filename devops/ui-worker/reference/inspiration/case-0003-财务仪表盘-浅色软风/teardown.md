# 财务仪表盘 · 浅色软风(暖调珊瑚橙) 拆解

> 来源：用户提供的参考图(Financial Dashboard,Dribbble 风) · 类型：Web 仪表盘 · 录入：2026-06-08
> 截图：把参考图存为本目录 `cover.png`。
> 学习目的：掌握"浅色 + 大圆角软卡片 + 单一暖强调色"这套亲和系仪表盘风格,补足深色之外的能力。

## 一句话为什么好看
大量留白 + 纯白大圆角卡片 + 唯一珊瑚橙强调，friendly、轻盈、不冷，是 fintech/SaaS 浅色仪表盘的高级范式。

## 视觉拆解
- **配色**：背景暖浅灰 `#ECECEC`；卡片纯白 `#FFFFFF`；主文本近黑 `#1A1A1A`、次文本灰 `#8A8A8A`；**唯一强调=珊瑚橙 `#EF5B2B`**（CTA、关键数字、环形进度、强调点）；主按钮用纯黑/橙两色 pill。强调色面积很小，全靠白底+留白撑高级感。
- **字体/排版**：干净无衬线、标题加粗友好；大字问候语"Hey, Need help? 👋 / Just ask me anything!"用近黑 + 浅灰双色对比。数字大而粗。
- **布局/栅格**：顶部条(logo№ / 头像+角色 / 搜索 pill) → 问候+任务行 → Bento 卡片网格。卡片大小不一、错落。
- **圆角/阴影/材质**：**超大圆角(卡片 ~24–32、按钮全 pill)**；阴影宽、淡、扩散(neumorphic-light)，几乎只靠极淡投影分层，不用边框。
- **图形语言**：同心半透明橙圆("年度利润 $4K→$14K")、环形进度("36% Growth rate")、蜡烛/竖条、点阵进度(13 Days)、笑脸评分(Review rating)、pill 下拉/筛选带 ×。
- **最妙细节**：① 同心圆叠数字表达"分层累积"很直观；② 笑脸评分把冷数据变亲和；③ pill 形态贯穿全局(按钮/下拉/标签)统一且软。

## 可复用要点（重点）
- **浅色仪表盘公式**：暖浅灰底 + 纯白大圆角卡 + 宽淡扩散阴影 + **唯一暖强调色**，瞬间 friendly 高级。
- **全 pill 化**：按钮/下拉/筛选/标签都用 pill，统一柔和气质。
- **强调色只点睛**：CTA、关键数字、进度环用橙，其余黑白灰，不滥用。
- **图形亲和化**：同心圆/笑脸/点阵 替代冷冰冰的纯数字。
- **大留白 + 错落 Bento**：卡片大小不一但对齐到网格，呼吸感强。

## 适用场景
fintech、SaaS、后台、个人财务/效率类的**浅色**仪表盘与首页。
