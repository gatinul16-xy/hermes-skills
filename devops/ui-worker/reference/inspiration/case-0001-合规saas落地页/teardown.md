## 一句话为什么好看
深色科技底 + 只用一个祖母绿强调色，靠"会扫描的实时演示卡"做钩子，专业、克制、有信任感。

## 视觉拆解
- **配色**：背景 #0a0e15 / 卡片 #121925 / 边框 rgba(255,255,255,.07) / 强调祖母绿 #34d399 / 文本 #eef2f7 / 次文本 #92a0b4。强调色面积 <10%，只落在 CTA、关键数字、状态。冷色基调 + 暖金少量。
- **字体 / 排版**：标题 Space Grotesk（紧字距、大字重），中文 Noto Sans SC，数字/标签 JetBrains Mono。H1 clamp 响应式、tracking 收紧；正文行宽限制、中灰色。
- **布局 / 栅格**：非对称 Hero（左文案右演示卡）；能力区用 Bento 非对称网格，不用三等分卡片；max-width 1180 居中。
- **层次手法**：靠字号/字重/色对比拉主次，不堆边框；一屏一个视觉重心。
- **间距节奏**：8pt 标度，区段大留白。
- **圆角 / 阴影 / 材质**：圆角统一（18/30）；阴影宽而淡、向背景色染（非纯黑）；演示卡轻玻璃感。
- **动效 / 微交互**：演示卡顶部"扫描光"循环；状态点呼吸；按钮 :active 下沉；滚动错落揭示；只动 transform/opacity。
- **最妙的细节**：① 用"受限方筛查/HS归类/危险品/准入拦截"四态真实演示卡当钩子，比堆字有说服力；② 数字用等宽体 + 暖金，单点睛；③ 失败/拦截态用语义红，但全局仍单强调色。

## 可复用要点（重点）
- 深色 B 端：**单一强调色 + 中性灰阶**，强调色只给 CTA/关键数字/状态，瞬间显专业。
- Hero 用**非对称 + 一张"会动的产品演示卡"**做钩子，胜过纯文案。
- 能力区**用 Bento 非对称网格代替三等分卡片**，更高级。
- 字体：display 用有性格的 grotesk（避开 Inter），数字用 mono，中文 Noto Sans SC。
- 阴影向背景色染、宽而淡；动效只动 transform/opacity。

## 适用场景
深色科技风的 B 端/SaaS/AI 产品落地页、控制台首页、发布页。
