## 一句话为什么好看
三台橙边手机并排放在奶油色背景上,屏内是干净的浅色 iOS 界面 + 柔和 pastel 统计卡,既展示了多屏流程,又 friendly、有呼吸感。

## 视觉拆解
- **展示排版**：**三台手机并排**(home/map、activity、profile),放在暖奶油色 `#F7ECE3` 大圆角面板上——这是 App 作品展示的经典排版,一屏讲完一条体验流程,也天然是横屏。
- **机身**：橙色机身边框(`#E8773A`→`#C75A28` 渐变),黑色 Dynamic Island,大圆角 ~46px。
- **屏内配色**：白底 `#FFFFFF` + 近黑文本 `#1C1C1E` + 灰次文 `#8A8A8E`;**单一橙强调** `#EF6C33`(机身/CTA/marker/激活态);统计卡用**柔和 pastel**:浅蓝/浅紫/浅青/浅粉,每类一色、低饱和。
- **组件**：pill chips(Walkie Talkie/Tracking/Geofence)、2×2 统计卡(图标+标题在上、大数字在下、"10% more than yesterday"小注)、Attention 提醒行、柱状图卡、资料行(Name/Breed/Weight 右对齐值)、底部 tab(一个深色实心圆激活,其余线性图标)。
- **地图**：柔和浅色地图——浅绿 park、浅蓝水塘、浅灰路网、虚线路径 + 头像 marker pin。
- **最妙细节**：① pastel 统计卡用颜色分类信息,清爽不花;② 头像 marker(宠物头像嵌进定位针)很可爱;③ 底部 tab 单个深色实心圆激活,识别清晰。

## 可复用要点（重点）
- **App 展示=三联手机并排 + 奶油大圆角底板**:讲多屏/流程的首选排版,也适配横屏。
- **浅色 iOS native 公式**:白卡 + 近黑/灰文 + 单一暖强调 + **柔和 pastel 分类卡**。
- **统计卡结构**:图标+标题(上)→ 大数字(下)→ 对比小注;每类一个低饱和底色。
- **底部 tab**:激活项用深色实心圆,其余线性,极易识别。
- **柔和地图**:浅绿/浅蓝/浅灰 + 虚线路径 + 头像 marker,轻量不喧宾夺主。
- **橙机身**:给手机框上品牌色边,整组更有调性。

## 适用场景
移动 App 的作品展示图、落地页 hero、App Store 截图、宠物/健康/活动/地图类浅色 App。
