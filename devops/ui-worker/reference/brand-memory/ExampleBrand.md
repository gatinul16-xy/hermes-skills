# ExampleBrand 品牌记忆

> 定位:示例 AI Agent 垂类。讲 AI 工作流、能力拆解、实战。
> 媒介:短视频(Remotion/真实录屏)、图文卡片、产品落地页/控制台。
> 更新:2026-06-07
> **权威源(数值以这些为准,本档是摘要 + 指向):**
> - 视频:`~/projects/video_design_system.md`
> - Remotion tokens:`~/remotion-base/src/theme.ts`

## 调性关键词
深色科技 · 克制专业 · 信息清晰 · 一眼看懂 · 高手感(不浮夸、不 PPT 感)

## Logo / 标识
- **AI 芯片 + 发光神经网络脑** 是唯一视觉标识。
- 文件:`remotion-base/public/brand/ai-chip-brain.png`(视频里低透明度背景资产,做呼吸微动,不抢主体)。
- 文字署名:`ExampleBrand`,可轻量出现在角落,不抢首屏钩子。

## 配色 token(深色科技 · 核心)
- 背景 `#06111f`(深空蓝黑) / 卡片 `rgba(8,18,38,0.82)` / 边框 `rgba(91,196,255,0.28)`
- 主文本 `#f8fafc` / 次文本 `#94a3b8` / 更弱 `#64748b`
- **强调主:蓝青高光 `#38bdf8` + 青 `#22d3ee`**;强调辅:紫 `#8b5cf6`(少量串联/连线)
- 电光蓝 `#2563eb`(网格/底层轮廓);状态:成功 `#22c55e` / 警告 `#f59e0b` / 错误 `#ef4444`
- **面积法则**:75% 深色底,蓝青高光约 10%,紫/状态色只点睛。禁满屏高饱和、禁纯黑 `#000`。

## 字体 / 排版
- 中文:**Noto Sans SC**(必须显式引入,Black/Bold/Medium/Regular);别依赖系统加粗。
- 等宽/英文数字/极客符号:**JetBrains Mono**(标签前可加 `[01]`、`</>`、`>>` 等高亮符号)。
- 圆角默认 `22`;一屏最多 1 主标题 + 3 短标签 + 1 行辅助说明。

## 版式习惯
- 三段式:顶部大标题 / 中央深色玻璃卡片承主信息 / 底部品牌。
- 背景只做氛围与识别(粒子/网格/品牌图呼吸),不抢主体;禁满屏字、复杂雷达图、PPT 卡片堆叠。
- 视频比例默认竖版 9:16;依赖录屏/横向对比时用 16:9。

## 动效
- 轻、快、有目的:入场 spring 180–320ms;旧元素被新元素挤开/缩小/降透明,不硬切叠。
- 镜头轻推近/平移,不大幅旋转/3D 翻牌;强调用框选/下划线/局部放大/数字跳动,不堆发光。
- 视频用 timings 驱动(口播到哪出哪),避免死静态(见 remotion-video 技能)。

## 各媒介具体值
- **视频(深色科技风,主)**:上面这套(theme.ts / video_design_system.md)。
- **图文卡片(VibeCoding 系列等)**:终端赛博变体——底 `#0a0e15`,青 `#22d3ee` + 暖黄 `#ffdd4d` + 紫点缀,标题 **Space Grotesk** + Noto Sans SC,等宽 JetBrains Mono,带 `{}` `</>` `>_` 终端角标。属同一品牌的"更亮、更梗"子风格,用于图文海报;视频仍以深色科技风为准。
- **落地页/控制台**:沿用深色科技 token(参考示例落地页范式:非对称 Hero + 单强调色 + 会动的演示卡)。

## 禁用(NG)
- 禁"AI 生成感/平庸 PPT 感";禁满屏文字、复杂信息图;禁纯黑底;禁高饱和乱配;
- 禁标识缺席(视频/封面应有 AI 芯片脑);禁中文不引 Noto Sans SC 导致渲染异常。
